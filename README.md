# homebrew-rosie

Homebrew tap for [rosie](https://github.com/matthewp/rosie) - a robot helper for agent skills.

## Installation

```bash
brew tap matthewp/rosie
brew install rosie
```

## Usage

```bash
rosie install owner/repo
rosie list owner/repo
rosie remove skill-name
rosie agents
```

See the [main repository](https://github.com/matthewp/rosie) for full documentation.
